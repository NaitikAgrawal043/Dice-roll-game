# 🎲 Dice Roll Game

A fun, beginner-friendly dice rolling game made with **HTML**, **CSS**, and **JavaScript**.

## 📌 What does it do?

- Simulates rolling two dice for **Player 1** and **Player 2**.
- Randomly displays dice faces (1–6) for each player.
- Shows who wins or declares a draw.
- Click **anywhere on the page** or the **button** to roll again.
